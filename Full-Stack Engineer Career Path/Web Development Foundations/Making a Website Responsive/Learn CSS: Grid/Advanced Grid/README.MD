Grid Template Areas
- The grid-template-areas property allows you to name sections of your web page
- e.g grid-template-areas: "header header"
                      - "nav nav" 
                      - "info services"
                      - "footer footer";
  - grid-template-rows: 300px 120px 800px 120px;
  - grid-template-columns: 1fr 3fr;
